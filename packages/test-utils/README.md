# @formkit-gov/test-utils

Testing utilities for forms built with FormKit Gov including render helpers, interaction utilities,
and accessibility testing.

## Status

**Coming in v0.2.0** - This package is currently a placeholder.

See the [roadmap](../../docs/ROADMAP.md) for planned features.

## Planned Features

- Form render helpers for React Testing Library
- VA Design System component interaction utilities
- Accessibility testing helpers (axe-core integration)
- Form submission mocking
- Validation testing utilities
- Snapshot testing helpers

## Installation

```bash
pnpm add -D @formkit-gov/test-utils
```

## Peer Dependencies

- `react` >= 17.0.0
- `react-dom` >= 17.0.0
- `vitest` >= 1.0.0 (optional)
- `@testing-library/jest-dom` >= 6.0.0 (optional)
