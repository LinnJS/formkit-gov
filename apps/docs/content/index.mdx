## Features

- **VA Design System** - Pre-built components wrapping official VA web components
- **Zod Validation** - Type-safe validation schemas for SSN, phone numbers, addresses, dates, and more
- **React Hook Form** - Seamless integration using the shadcn/ui composable pattern
- **Accessibility First** - WCAG 2.1 AA compliant with keyboard navigation and screen reader support

## Installation

```bash npm2yarn
npm install @formkit-gov/react @formkit-gov/core
```
