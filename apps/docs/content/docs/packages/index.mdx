# Packages

FormKit Gov is organized as a monorepo with specialized packages for different concerns.

## Core Packages

| Package                                    | Description                                | Status |
| ------------------------------------------ | ------------------------------------------ | ------ |
| [@formkit-gov/core](/docs/packages/core)   | Zod schemas, validators, and patterns      | Stable |
| [@formkit-gov/react](/docs/packages/react) | React components wrapping VA Design System | Stable |

## Package Dependencies

```text
@formkit-gov/react
  └── @formkit-gov/core (peer dependency)
      └── zod (peer dependency)
```

All packages are designed to work together but can be used independently based on your needs.
